# Stockbay

Stockbay is a web application for managing stock for supply stores

## Information:
- Status: under development
- latest version 1.0
- Industry sector
- Created: November 2020
- Last updated: December 2020

## Modules:
- Manage products by categories
- Manage purchases
- Manage sales
- Classify suppliers
- Manage clients
- Manage employees
- Manage income
- Manage expenses
- Manage inventory
